# SiCATA-SistemInformasiCatatanAdministrasi
SiCATA adalah aplikasi web manajemen surat masuk dan keluar yang dirancang khusus untuk digitalisasi administrasi perkantoran desa. Sistem ini mengotomatiskan pembuatan nomor surat, menyediakan arsip digital yang mudah dicari, serta menghasilkan dokumen Word dengan format kop surat resmi secara instan.
